# react-context-todo-app
Todo App built using React and the Context API

## Install

Install with npm:

```bash
npm install
```

Install with yarn:

```bash
yarn install
```

## Run

Run with npm:

```bash
npm start
```

Run with yarn:

```bash
yarn start
```

## Build

build with npm:

```bash
npm run build
```

Run with yarn:

```bash
yarn build
```
